What is PitWall AI
The Fan Intelligence Engine

There are 500 million F1 fans worldwide. Every race weekend, they watch without knowing what's actually happening inside the car.

Sky Sports tells you what happened. PitWall AI tells you why — using the same telemetry signals real F1 engineers use on race day.

We built a Driver DNA fingerprint for each driver using unsupervised machine learning. When a driver deviates from their own baseline, our system flags it in real time.

The Technology
How It Works
01
Data Pull
Real telemetry pulled from the official F1 API via FastF1 — throttle, brake, speed, RPM, gear, DRS across every lap.
02
DNA Build
StandardScaler learns what "normal" looks like for each driver individually — their throttle fingerprint, their braking signature.
03
Anomaly Flag
Isolation Forest compares every lap against the driver's own baseline. Deviating laps get flagged with a severity score.
04
AI Radio
Groq's Llama 3.3 translates every anomaly into a plain-English race engineer radio message — calm, professional, real.
